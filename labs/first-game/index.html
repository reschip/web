First Game with raylib

In this lab, I’ll document my experience creating my first game using the raylib library, a simple yet powerful tool for game development.

What is raylib?

raylib is a game development library focused on learning to program games, with a simple and easy-to-use syntax. It’s perfect for beginners.

Game Concept

My first game is a simple 2D platformer where the player must collect objects while avoiding obstacles. The goal was to learn the basics:

  • Sprite rendering
  • Collision detection
  • Basic physics (gravity and jumps)
  • User input
  • Game state management

Featured Code

// raylib initialization
InitWindow(screenWidth, screenHeight, "My First Game");

// Main game loop

Lessons Learned

Creating this first game has been a great learning experience about code organization, game lifecycle, and the importance of optimization.
